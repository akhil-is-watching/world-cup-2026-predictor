# FIFA World Cup 2026 Winner Prediction Model 🏼⚽

An AI-powered prediction model and interactive web application that forecasts the winner of the FIFA World Cup 2026.

## 🎯 Project Overview

This project combines:
- **Python Prediction Model**: Machine learning-based analysis of team strengths
- **React Web Application**: Interactive visualization and exploration of predictions
- **Vercel Deployment**: Live, scalable hosting

## 📈 Prediction Model

The model analyzes multiple factors:
- **FIFA Ranking** (35%) - Official international rankings
- **Recent Form** (25%) - Team momentum and performance  
- **Squad Strength** (20%) - Player quality assessment
- **Team Experience** (15%) - Tournament experience
- **Confederation Bonus** (5%) - Regional performance trends

## 🌐 Website Features

- **Interactive Charts**: Bar and pie charts showing probability distributions
- **Team Rankings**: Filterable and sortable table of all 48 teams
- **Confederation Filter**: View predictions by continental confederation
- **Team Details**: Click on any team for detailed statistics
- **Responsive Design**: Works seamlessly on desktop and mobile
